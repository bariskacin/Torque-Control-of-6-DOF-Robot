# Torque Control of 6-DOF Robot

This project includes torque control and simulation of a 6-degree-of-freedom robot. Matlab files, SolidWorks models, and simulation results are provided in this repository.

## Folders
- `Matlab/`: Matlab codes, Simulink files, and data files
- `sldprt/`: SolidWorks part and assembly files
- `step/`: 3D models in STEP format

## Usage
1. Use Matlab or Simulink to run the Matlab files.
2. Open SolidWorks files with SolidWorks or a compatible CAD program.
